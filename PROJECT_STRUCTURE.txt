QUICK START
===========

1. Install:     pip3 install -r requirements.txt
2. Configure:   Edit config.yaml with your API key
3. Verify:      python3 test_setup.py
4. Run:         python3 book_tracker.py scan


COMMANDS OVERVIEW
=================

Scanning:
  scan                  - Process images in directories
  
Entry:
  add                   - Add book manually
  
Viewing:
  list                  - List all books
  show                  - Show detailed book info
  search                - Search with filters
  
Management:
  update                - Update book details
  enrich                - Fetch/refresh metadata
  
Categories:
  categories list       - Show predefined categories
  categories add        - Add new category
  categories remove     - Remove category
  
Export:
  export                - Export to CSV or JSON
